NFC Tags for Descenders

Digitally track, inspect, and manage descenders and other hardware and devices using durable NFC tags and the Scannable app.

Descender NFC tags designed with durability in mind, built to withstand the harshest environments.

Glue-On NFC Tag for Descenders

  • The world's smallest NFC tag!
  • This tag is low profile and has no hard edges
  • Place in a recess or a low wear area
  • These tags only read if placed on metal
  • These tags need to be applied with an adhesive. Check out this blog for our adhesive recommendations
  • Available in Lime for those who like to stand out, or Grey if you need a something a little more stealthy
Tag application and placement guide
Get the tag

This tag works well on carabiners like: • Petzl Ascention • Petzl Rig • Petzl I’D • Petzl Pro Traxion • Safe-Tec Duck • Skylotec Sirius • Edelrid Hand Cruiser

Self-adhesive NFC Tag for Descenders

  • High-tack self-adhesive NFC sticker
  • Use in well-protected, low-impact areas
  • Avoid placing on edges, corners, or areas that can take knocks
  • Do not place in a proud area
  • Consider using the Carabiner/Hardware tag or Rope Tag for a more robust solution
